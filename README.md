# AIに依存するかゲーム

単一HTMLで作成したゲームプロトタイプです。

## 概要

AIを使うことで作業効率は上がるが、その一方で「自分で考える力」が徐々に低下していく——
そんな現代的なトレードオフを体験できる意思決定ゲームです。

プレイヤーは各ターンごとに、

* AIに頼る
* 自分で考える
* 使い分ける（一定の思考力がある場合）

の選択を行いながら進行します。

選択によって「思考力」「ストレス」「AI依存度」が変化し、
最終的な状態によって複数のエンディングに分岐します。

思考力が低下すると、

* AIの出力の質を見抜けなくなる
* 判断の精度が下がる
* 選択の自由度が制限される

など、プレイ体験そのものが変化します。

## 遊び方

1. ページを開く
2. 各ターンで表示される状況に対して選択肢を選ぶ
3. ステータス（思考力・ストレス・AI依存度）の変化を確認する
4. 最終ターンまで進めるとエンディングが表示される

## 目的

このプロトタイプで検証したかったこと：

* AIを使うことによる「短期的な効率」と「長期的な思考力低下」のトレードオフ
* 思考力が低下したときに起きる「判断の質の変化」
* 能力によって「選択肢の見え方」が変わる構造
* AIを「代行」として使うのか、「補助」として使うのかによる結果の違い

また、ゲームという形式を通じて
「AIとの適切な距離感」を体験的に理解できるかを検証しています。
